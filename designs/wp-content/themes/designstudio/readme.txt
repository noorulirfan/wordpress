== Description ==

DesignStudio is fully Responsive Design WordPress Theme. Features: Animation Appearing Effects, Custom Logo, Custom Background, Footer copyright text customizations, Widget Areas: Sidebar, 3 Homepage Columns, 3 Footer Columns, Instagram Widget support, Facebook Like Box Widget Support, Tabs Widget support, Footer Menu, Translation-Ready and much more.
